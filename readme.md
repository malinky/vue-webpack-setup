## Vue + Webpack Basic Setup

Basic Vue set up with webpack, vue-loader and vue-hot-reload-api

#### Setup

* npm install
* webpack (Build public/app.js)
* webpack-dev-server --inline --hot (Start server and build public/app.js with hot reloading)
